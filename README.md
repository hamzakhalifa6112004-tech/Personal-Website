This project is a personal website built as part of a **Frontend Mentor challenge**. It is designed to showcase responsive layouts, modern CSS techniques, and best practices in frontend development.

## Overview

The goal of this project was to create a personal portfolio website that is **fully responsive** and demonstrates proficiency in HTML, CSS, Flexbox, Grid, and Sass.  

The project includes sections for:  

- Personal introduction  
- Portfolio or projects  
- Contact information  
- Responsive design for desktop, tablet, and mobile  

---

## Technologies

This project uses:  

- **HTML5** – Semantic markup for structured content  
- **CSS3** – Styling and responsive design  
- **Sass (SCSS)** – Modular and maintainable styles  
- **Flexbox** – Layout and alignment of elements  
- **CSS Grid** – Advanced layout for sections and grids  
- **Live Server** – For local development and real-time preview  

---

## Features

- Fully responsive design using **Flexbox** and **Grid**  
- Modular CSS using **Sass partials and variables**  
- Smooth hover effects and transitions  
- Semantic and accessible HTML structure  
- Mobile-first design approach

## Installation

1. Clone the repository:  

```bash git clone <your-repo-url>```
Navigate to the project folder:

```bash cd personal-website```
Install Sass globally (if not installed):

```bash npm install -g sass```
Compile SCSS to CSS:

```bashsass scss/style.scss css/style.css --watch ```
Open index.html in your browser or use Live Server in VS Code.

Usage
Open index.html to view the website

Edit SCSS files in the scss folder

Changes are automatically compiled to CSS when using --watch
